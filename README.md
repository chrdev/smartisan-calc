# smartisan-calc

Smartisan Calculator 8.1.0 Android APP MOD

[English README](README_enu.md)

锤子计算器 基于 [SmartisanOS_APP_Port](https://github.com/People-11/SmartisanOS_APP_Port) 修改。

感谢 [People-11](https://github.com/People-11) 的工作！

## 快速体验

复制下面行，双击文字区域，将在竖屏得到计算结果 0.7734。

```
c[rad]2xcos(π/3)-0.4532÷2=!p
```

## 相对于原版的变化

1. 原版的 AC 等同 C。 此版本 AC 等于 C + MC。

2. 长按文字直接复制结果，不再显示 Copy 菜单。

3. 在文字区域可实现粘贴，切换横竖屏，调整按键音等功能。文字区域的动作响应范围是显示两行文字的部分。竖屏时在文字上方可能存在很大空白，此空白部分是不响应操作的。

## 使用说明

### 复制

长按文字区复制。

### 粘贴

快点两下文字区粘贴，本质是把剪贴板的文字解析为按键依次发送。现在可以在便签写下计算式，然后粘贴到计算器。

在竖屏也可粘贴横屏科学计算器的按键，大小写皆可，空白和换行会被忽略。多字按键的通用规则是用方括号`[]`包裹按键字样。

以`!`开头的不是按键，而是命令。执行命令后不会继续解析剪贴板，所以应该单独使用，或做为最后一项。

比如：

```
[AC]125[Del][+/-]!L
```

执行结果是全清后输入`-12`，并转横屏。因为其中`5`被`[Del]`删除了，然后由`[+/-]`变负号，最后由`!L`命令转为横屏。

由于一些手机键盘输入方括号比较繁琐，也可用一对反斜杠`\\`来包裹长按钮。

部分常用键可以免包裹符号，见下表。

### 切换横竖屏

原版需要手机开自动旋转才能切换横竖屏，现在可通过手势或命令来切换。
	
#### 手势切换

在文字区以水平方向抛掷，跨度超过屏幕水平方向宽度 80% 。如果不清楚抛掷动作，请看后面说明。

#### 命令切换

粘贴命令，不分大小写。

```
!L : 横屏。
!P : 竖屏。
!R : 切换屏幕方向。
!o : 切换屏幕方向，同 !R。
```

#### 关于抛掷

抛掷操作是，手指在屏幕划过一段距离，并在动作尾声以较快速度离开屏幕。
为了可靠的做出抛掷动作来切换横竖屏，请尝试下面的方式：

1. 在文字区靠左边缘的部分开始接触屏幕；
2. 手指不离屏幕，以任意速度，向右移动至距离屏幕右边缘约 2cm 处，即便上下偏斜了也没关系，只要距离右边缘 2cm 即可；
3. 手指继续不离开屏幕，快速向右划出屏幕之外。

这样应该就可以了。

抛掷只与手指开始接触屏幕的那一点，以及最后离开屏幕的点位置、速度有关，而与手指在屏幕上的具体轨迹无关。

### 调整音量

粘贴命令。

```
!0 : 关按键音
!1 : 小按键音（作者最爱）
!2 : 原版按键音
```

### 获取帮助

粘贴命令`!h`，会复制简要帮助内容到剪贴板，请在其它 APP 粘贴查看。
